Exercise 6. Change group license assignments

Learning path 2.: AZ-104: Manage identities and governance in Azure
Module: 2. Create, configure, and manage identities


Overgeslagen
Deze exercise vereist toegang tot de Microsoft 365 admin center > Billing > Licenses voor het toewijzen van licenties aan groepen. Dit vereist een zakelijke Microsoft 365 subscription.
Met een persoonlijk Azure account is deze pagina niet toegankelijk:

"Your account doesn't have permission to view or manage this page. Login is not supported for consumer users without business presence."

In de Microsoft Entra admin center is navigeren naar Groups > [groep] > Licenses wel mogelijk — hier is de huidige licentiestatus zichtbaar, maar wijzigingen vereisen de Microsoft 365 admin center.
