# Exercise 4. Restore or remove deleted users

- Learning path 2.: AZ-104: Manage identities and governance in Azure
- Module: 2. Create, configure, and manage identities

In deze oefening verwijder ik een gebruiker uit Microsoft Entra ID en herstel ik deze daarna via de Deleted users lijst.

---

## Task 1 Gebruiker verwijderen

Gebruiker ChrisG was al verwijderd in Exercise 3 — deze stap was daardoor al uitgevoerd.

Verwijderen gaat via **Microsoft Entra admin center > Identity > Users > [gebruiker selecteren] > Delete user**.

---

## Task 2 Verwijderde gebruiker herstellen

Uitgevoerd via **Microsoft Entra admin center > Identity > Users > Deleted users**

- Gebruiker Chris Green geselecteerd uit de lijst
- Hersteld via **Restore user**
- Gecontroleerd via **All users**: Chris Green staat er weer bij
